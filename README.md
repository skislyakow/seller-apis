# Скрипты для автоматического обновления ассортимента электронной витрины на Озон и Яндекс-Маркет
## Скрипт market.py для Яндекс-Маркет
## Скрипт seller.py для Озон
### Описание функционала
Скрипты позволяют автоматизировать следующие действия:
1. Получить артикулы товаров с маркеплейса
2. Получить  текущий ассортимент с сайта casio
3. Обновить ассортимент и цены на витринах маркетплейсов
