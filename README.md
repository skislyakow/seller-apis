# Скрипты для работы с Озон и Яндекс.Маркет

Скрипты автоматически обновляют ассортимент и цены на маркетплейсах.

## Установка

1. Убедитесь, что установлен Python 3.8+.
2. Создайте виртуальное окружение:
   - Linux/macOS: `python3 -m venv venv; source venv/bin/activate`
   - Windows: `python -m venv venv; .\venv\Scripts\activate`
3. Установите зависимости: `pip install -r requirements.txt`

## Конфигурация

Создайте файл `.env` (или используйте `.env.example` как шаблон) со следующими переменными:

| Переменная | Описание |
|-----------|-----------|
| `MARKET_TOKEN` | Токен Яндекс.Маркет |
| `FBS_ID` | Идентификатор кампании FBS |
| `DBS_ID` | Идентификатор кампании DBS |
| `WAREHOUSE_FBS_ID` | Склад FBS |
| `WAREHOUSE_DBS_ID` | Склад DBS |
| `SELLER_TOKEN` | Токен Озон (seller) |
| `CLIENT_ID` | ID клиента Озон |

## Запуск

- Яндекс.Маркет: `python market.py`
- Озон: `python seller.py`

## Структура проекта

```
├── market.py       # Интеграция с Яндекс.Маркет
├── seller.py      # Интеграция с Озон
├── requirements.txt
└── .env.example
```

## Тестирование

Рекомендуется настроить `pytest` + `requests-mock` для изоляции внешних вызовов и проверки контракта API.

## Примечания

- Загрузка переменных окружения выполняется в `main()` через `Env()`, а не внутри функций — это соответствует лучшим практикам.
- Единая конвенция именования переменных и типов используется как в README, так и в коде.
